# Detetive dos Números - Projeto Acadêmico em Godot

Projeto em **Godot 4** desenvolvido como entrega de faculdade, com foco em jogo educativo infantil, progressão matemática e apresentação visual amigável.

## Recursos implementados
- nome personalizado do jogador
- tela de jornada em modal
- múltiplas fases com alternativas
- timer com urgência nos últimos segundos
- pontuação baseada em tempo de resposta
- badges e presente de recompensa
- chefão final com vidas
- pausa com opções completas
- fogos na celebração final
- sons gerados dentro do próprio projeto

## Estrutura
- `project.godot`
- `scenes/main.tscn`
- `scripts/main.gd`
- `data/game_content.json`
- `docs/GDD.md`

## Como abrir
1. Abra o Godot 4.
2. Importe `project.godot`.
3. Rode com `F5`.

## Observação
Os arquivos de texto do projeto foram mantidos em UTF-8.
